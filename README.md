# NotesApp
## Обязательные и Желательные требования к тестовому выполнены. Также есть возможность прикрепить фотографию в заметку. 
Комментарий к тестовому заданию:
Выбрал архитектуру CleanSwift для будущей расширяемости приложения и максимальной чистоты модулей, а также в данной архитектуре легче проводить тестирование.
С точки зрения расширяемости это будет хороший вариант.


Core Data выбрана как стандартная БД из "коробки"
Интерфейс через IB, знаю что это не самая предпочитаемая практика для возможных сложных интерфейсов и расширения, но в данном случае использовал для увеличения скорости разработки.

## Недоработки
Фото можно прикрепить, а вот сохранить нет. Не успел реализовать сохранение фотографии в заметке на том месте, где она была поставлена.
Не успел реализовать выделения и типы шрифтов для выделенного текста.
Можно было изначально сделать VC для изменений и создания заметки одним и тем же, только наполнять данными по разному, это тоже не успел переделать.
