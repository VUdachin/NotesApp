# NotesApp
Обязательные и Желательные требования к тестовому выполнены. 
Комментарий к тестовому заданию:
Выбрал архитектуру CleanSwift для будущей расширяемости приложения и максимальной чистоты модулей, а также в данной архитектуре легче проводить тестирование.
С точки зрения расширяемости это будет хороший вариант.


Core Data выбрана как стандартная БД из "коробки"
Интерфейс через IB, знаю что это не самая предпочитаемая практика для возможных сложных интерфейсов и расширения, но в данном случае использовал для увеличения скорости разработки.
